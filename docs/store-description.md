## Full Description (4000 characters max)

10K Scorekeeper is the ultimate companion app for the classic dice game 10,000 (also known as Farkle, Greed, or Ten Thousand).

**Stop using paper scoresheets!** Our app makes it easy to track scores, see who's winning, and keep a history of all your games.

### KEY FEATURES

**Easy Score Tracking**
- Quick-add buttons for common scores (50, 100, 150, 200, etc.)
- Custom score entry for any amount
- Track multiple players across unlimited rounds
- Red "bust" indicators show when someone rolled too risky

**Multiplayer Games**
- Create games with 2-10 players
- Share game codes to invite friends
- Add guest players who don't have the app
- Real-time score updates across devices

**Statistics & History**
- Track your wins, games played, and win percentage
- See your best score and average score per game
- View your best single turn and average per round
- Browse completed games with full score breakdowns

**Beautiful Design**
- Clean, easy-to-read scorecard layout
- Dark and light theme options
- Works great on phones and tablets

### HOW IT WORKS

1. Create an account (email or Google sign-in)
2. Start a new game or join with a game code
3. Add players and start scoring
4. Tap quick-add buttons or enter custom scores
5. First to 10,000 wins!

### ABOUT THE GAME

10,000 (Ten Thousand) is a classic dice game played with six dice. Players take turns rolling to accumulate points, but risk losing their turn's points if they "bust" by rolling no scoring dice. The first player to reach 10,000 points wins!

Common names for this game include:
- Farkle
- Greed
- Ten Thousand
- Zilch
- Dice 10000

### WHY 10K SCOREKEEPER?

- **Free to use** - No ads, no premium tier
- **Works offline** - Scores sync when you're back online
- **Privacy focused** - We don't sell your data
- **No account required for guests** - Only the game creator needs an account
